# Custom devices (DiY) ptvo.switch

| Model | ptvo.switch  |
| Vendor  | Custom devices (DiY)  |
| Description | [Multi-channel relay switch](https://ptvo.info/zigbee-switch-configurable-firmware-v2-210/) |
| Supports | hold, single, double and triple click, on/off, type, rssi |
| Picture | ![Custom devices (DiY) ptvo.switch](../images/devices/ptvo.switch.jpg) |

## Notes

The following page describes all possible topics and commands, but it may depends on your firmware configuration.

## Manual Home Assistant configuration
Although Home Assistant integration through [MQTT discovery](../integration/home_assistant) is preferred,
manual integration is possible with the following configuration:


{% raw %}
```yaml
switch:
  - platform: "mqtt"
    state_topic: "zigbee2mqtt/<FRIENDLY_NAME>"
    availability_topic: "zigbee2mqtt/bridge/state"
    payload_off: "OFF"
    payload_on: "ON"
    value_template: "{{ value_json.l1 }}"
    command_topic: "zigbee2mqtt/<FRIENDLY_NAME>/l1/set"
```
